# Student Score Analyzer 📊

Aplikasi sederhana berbasis Python untuk menganalisis nilai siswa secara otomatis.

## Fitur Utama
* Menghitung rata-rata nilai dari berbagai mata pelajaran.
* Menentukan Grade (A, B, C, D) secara otomatis.
* Mengurutkan ranking siswa dari yang tertinggi.
* Export hasil analisis ke dalam file CSV (`result.csv`).

## Cara Penggunaan
1. Pastikan sudah install Python dan library Pandas (`pip install pandas`).
2. Masukkan data siswa di `students.csv`.
3. Jalankan `python main.py`.
